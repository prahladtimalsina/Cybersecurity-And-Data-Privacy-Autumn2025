# Privacy Policy – Booking System

## 1. Introduction
This Privacy Policy describes how we collect, use, store, and protect personal data in our booking system. The system is designed following the principles of Privacy by Design and the requirements of the EU General Data Protection Regulation (GDPR).

## 2. Data Controller

## 3. Personal Data We Collect
We process the following personal data:


| Category          | Data                     | Purpose                          | Lawful basis (GDPR       |
|-------------------|--------------------------|----------------------------------|----------------------------|
| Account data      | Name, email, password, date of birth | Account creation & age verification | Contract (Art. 6(1)(b)) |
| Booking data      | Resource ID , Reservation history, time slot   | Provide the booking service      | Contract (Art. 6(1)(b)) |
| Technical data    | IP address, browser, logs| Security and debugging           | Legitimate interest (Art. 6(1)(f)) |

We do **not** publicly display the identity of reservers.

## 4. Legal Basis for Processing
We process data based on:
- **Contract** – providing the booking service.
- **Legitimate Interest** – maintaining security and preventing abuse.
- **Consent** – when required (e.g., for non-essential cookies).

## 5. Purpose of Processing
Personal data is used to:
- Create and manage user accounts
- Verify age eligibility (15+)
- Manage reservations
- Provide administration functions
- Improve system security
- Comply with legal obligations

## 6. Data Retention
Personal data is stored only as long as required:
- User account data: until account deletion
- Reservation data: as required for service records
- Logs: max 12 months

## 7. User Rights
Users have the following GDPR rights:
- Right of access
- Right to rectification
- Right to erasure (“right to be forgotten”)
- Right to restrict processing
- Right to object
- Right to data portability
- Right to lodge a complaint with a supervisory authority

Requests can be sent to the contact email above.

## 8. Cookies
We use essential cookies for login and session management.  
Non-essential cookies are used only with explicit consent.

See our Cookie Policy for more details.

## 9. Data Security
We implement:
- Secure HTTPS connections
- Encrypted passwords
- Role-based access controls
- Logging for suspicious activity
- Regular security reviews

## 10. Data Sharing
We do not share personal data with third parties unless required by law or specified in this policy.
